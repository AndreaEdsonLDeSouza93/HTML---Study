images types and fonts

  1. RASTER: pixels
  2. VECTOR: matematichal vectors like lines and curves

we use raster images for photos and vector images for simple graphics, icons, logos and so on

Raster Image

  • Come from cameras/scanners (but they can be also produces by software)
  • Formats: JPG, PNG, GIF, etc
  • More pixels === larger file size
  • Look blurry if scaled up

Vector Image

  • Created in software (Adobe Illustrator)
  • Format: SVG
  • Scalable Vector Graphics
  • Look sharp at any size
  • They are "not made of pixels", they are defines by Matetichal vectors.

block level elements have by default 0 (height), and they strech only as much as it is needed, to container the elements.

body tag is a block level element.

For example, if we set the container's height to 20vh (with display set to block) and then set the height to 100%, it will completely fill the container. 

      
        body {
          background: url(../etc..);
          background-repeat: no-repeat;
          background-size: cover; /* cover all the background while keeping the aspect ratio. */
          height: 100vh;
        }

background-attachment: fixed; height: 300vh;

This will ensure that the background remains fixed, even as you scroll extensively.
