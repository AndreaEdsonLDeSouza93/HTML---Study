2-The Head Section

    head
  • charset="UFT-8" .. for exampla A === 65.
  • ASCII - American Standard Code for Information Interchange.
  • <meta charset="UTF-8">: This sets the character encoding for your HTML document to UTF-8. UTF-8 supports a wide range of characters from different languages, making sure that text is displayed correctly.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: This meta tag controls the layout on mobile browsers. It ensures that your website is responsive by setting the viewport width to the device's width and the initial zoom level to 1.0.
  • <meta name="keywords" content="HTML, CSS">: This meta tag contains keywords related to the content of your website. These keywords help search engines understand what your website is about and can improve your site's search engine optimization (SEO).
  • <meta name="description" content="what it is written here, it will appear as text / descrpt. of the site on google.">: This meta tag provides a brief description of your website's content. The description might appear in search engine results, giving users an idea of what your site is about before they click on the link.
