General rule: ID > Class > Tag

Example -> #product > .products -> ul

We can combine multiple selectors.

    0,0,0 is the point system
  • first 0 is ID
  • second 0 is Class
  • third 0 is Tag
  • Practicle example:
  • .products#product > #product
  • 110 > 100
  • h2.section.title > h2
  • 011 > 001
  • Finally you can use the keyword !important if you want to fix that rule and make it permanent.
