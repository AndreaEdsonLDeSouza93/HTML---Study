Key Features of Flexbox

  • Ease of Use: Simple properties like justify-content and align-items make layout adjustments quick and easy.
  • One-Dimensional Layout: Ideal for laying out items in a single direction (row or column).
  • Responsive Design: Flexbox adapts gracefully to container size changes.
  • Browser Support: Supported by all modern browsers, making it reliable for production.
  • Alignment Options: Align items vertically, horizontally, or both with minimal effort.
  • Reordering: Items can be easily reordered using the order property.
  • Dynamic Spacing: Use flex-grow, flex-shrink, and flex-basis for dynamic and proportional layouts.

Example 1: Centering a Div

This is the most common use case for Flexbox. Both vertical and horizontal centering are achieved easily:

Centered
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 200px;
}
