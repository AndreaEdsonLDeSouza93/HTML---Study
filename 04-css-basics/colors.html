Color Format Explanation

Explanation of Each Color Format:

Hexadecimal (HEX):

Format: #RRGGBB (each pair represents red, green, and blue intensities in hexadecimal notation).

Example: #3498db corresponds to:

  • Red: 34 in hexadecimal = 52 in decimal.
  • Green: 98 in hexadecimal = 152 in decimal.
  • Blue: db in hexadecimal = 219 in decimal.

HEX colors range from #000000 (black) to #FFFFFF (white).

RGB (Red, Green, Blue):

Format: rgb(r, g, b) where r, g, and b are integers between 0 and 255.

Example: rgb(255, 99, 71) means:

  • 255 (maximum red intensity),
  • 99 (medium green intensity),
  • 71 (medium-low blue intensity).

The result is a tomato-like color.

HSL (Hue, Saturation, Lightness):

Format: hsl(h, s, l) where:

  • h = Hue (angle on the color wheel: 0° is red, 120° is green, 240° is blue).
  • s = Saturation (0% is grayscale, 100% is full color intensity).
  • l = Lightness (0% is black, 50% is normal, 100% is white).

Example: hsl(120, 100%, 50%) is pure green (120° hue), fully saturated and with normal brightness.

Named Colors:

CSS supports a list of predefined color names, like coral, salmon, navy, etc.

Example: coral is a light reddish-orange color.

RGBA (RGB with Alpha):

Format: rgba(r, g, b, a) where a is the alpha channel (opacity, from 0.0 to 1.0).

Example: rgba(255, 0, 0, 0.5) gives semi-transparent red.

General Tips to Understand HEX and RGB Colors:

HEX to RGB Conversion:

Each HEX pair (e.g., 34 for red) represents a value from 00 (0) to FF (255).

To calculate manually, convert the hexadecimal value to decimal:

34 in HEX → (3 × 16) + 4 = 52 in RGB.

RGB to HEX Conversion:

Reverse the process: Convert each RGB value to hexadecimal.

Example: rgb(52, 152, 219):

  • 52 → 34
  • 152 → 98
  • 219 → db

Result: #3498db.

HSL to RGB Conversion:

This involves mathematical transformations (more complex). Use online tools or libraries for accurate conversions.

Color Selection:

Tools like Coolors or Adobe Color can help visualize and pick harmonious color palettes.

Experiment with varying saturation and lightness in HSL for pleasing effects.
