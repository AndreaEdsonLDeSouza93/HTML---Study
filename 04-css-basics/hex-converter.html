Letters in HEX Color Codes:

In a HEX color code (e.g., #RRGGBB), the letters used are A, B, C, D, E, and F. These letters represent values in hexadecimal, a base-16 numbering system that includes both numbers (0–9) and letters (A–F).

Value Mapping:

  • A = Decimal 10
  • B = Decimal 11
  • C = Decimal 12
  • D = Decimal 13
  • E = Decimal 14
  • F = Decimal 15

HEX combines these with numbers to represent a range from 00 (0 in decimal) to FF (255 in decimal) for each color channel (Red, Green, and Blue).

HEX to RGB Conversion:

A HEX color code is structured as #RRGGBB. Each pair represents the intensity of Red, Green, or Blue (ranging from 0 to 255 in decimal).

To convert:

  1. Split the HEX code into RR, GG, and BB.
  2. Convert each pair from hexadecimal to decimal.

Example Conversion:

HEX: #34A1F2

  • RR = 34 → (3 × 16) + 4 = 52 (Red = 52)
  • GG = A1 → (10 × 16) + 1 = 161 (Green = 161)
  • BB = F2 → (15 × 16) + 2 = 242 (Blue = 242)

Resulting RGB: rgb(52, 161, 242)

RGB to HEX Conversion:

An RGB value looks like rgb(r, g, b), where each channel ranges from 0 to 255.

To convert:

  1. Divide each channel by 16 to find the first HEX digit (quotient).
  2. Use the remainder for the second HEX digit.

Example Conversion:

RGB: rgb(244, 164, 96)

  • Red: 244 → 244 ÷ 16 = 15 (F) and remainder 4 → F4
  • Green: 164 → 164 ÷ 16 = 10 (A) and remainder 4 → A4
  • Blue: 96 → 96 ÷ 16 = 6 and remainder 0 → 60

Resulting HEX: #F4A460
