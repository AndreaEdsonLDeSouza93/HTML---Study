Type

Example, if you use an html element like body, p, div etc..

ID

You have an element <p id="text"> etc...

You can select that element and modify the text inside selecting the ID

Here's how you do it -> #text { css rules }

You can have only one ID per element. You can't have two identical ID in two differents elements

Class

You can have multiple elements with the same class.

example of class <p class="text"> etc...

Here's how you do it -> .text { css rules }

Attribute

google

Selectiong an element using an attribute is not very common.

Here we have a link with the target attribute -> target="_blank"

Here's how you do it -> a[target] { css rules } or a[target="_blank"] { css rules }

Another example would be a[href="https://google.com"] { css rules }

a[href*="google"]{ css rules } -> it checks for every a href with the word google in it.

this checks if a value stars with some particular value -> a[href^="https"]{ css rules }

href$="if_it_end_with_the_value_between_double_quotes"

We can combine multiple conditions.

Example: a[href^="https"][href$=".com"]
