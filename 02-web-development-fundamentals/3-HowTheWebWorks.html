  • HTTP
  • HTTPS: HTTP + Encryption

Accept-Language: en-us

  2. HTTP Response..

DOM: Document Object Model

Basically all the elements in our HTML document. The building blocks of our page like p, links, imgs etc..


<!DOCTYPE html>
<html>
...
</html>

If there are others url inside our dom, those url that will trigger a request towards those resources.

Once those calls (http request) has been made, the browser will render the HTML document.

  1. BROWSER send an HTTP request to the server.
  2. The BROWSER receive an HTTP response.
  3. The response contains an HTML Document.
  4. The BRO-WSER reads the HTML Doc. and constuct with it the DOM, after it will render the page.
