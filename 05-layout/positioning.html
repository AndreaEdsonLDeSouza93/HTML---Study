Relative Positioning

Relative positioning allows you to move an element relative to its original (static) position. The movement is achieved using top, left, right, or bottom. The element still occupies space in the normal flow.

Absolute Positioning

Absolute positioning positions an element relative to its nearest positioned ancestor. If no ancestor is positioned, it will be relative to the viewport. This removes the element from the normal flow, so other elements behave as if it doesn’t exist.

Absolute: Right 0, Bottom 0

Fixed Positioning

Fixed positioning positions the element relative to the browser’s viewport. It stays in the same place even if the page is scrolled. This is often used for sticky navigation bars or other UI elements.

Summary

Relative: Moves the element relative to its original position but doesn’t remove it from the flow.
Absolute: Positions the element relative to the closest positioned ancestor, removing it from the normal flow.
Fixed: Positions the element relative to the viewport, and it remains fixed while scrolling.
