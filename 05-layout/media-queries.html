What Are Media Queries?

CSS Media Queries allow you to apply different styles depending on the device‚Äôs screen size, resolution, or other properties. This is essential for responsive design, making websites look good on mobile, tablet, and desktop.

üéØ Syntax of a Media Query

  1. The @media rule.
  2. A condition (e.g., screen width).
  3. CSS rules that apply only when the condition is met.
@media (max-width: 768px) { body { background-color: lightblue; } }

‚úÖ If the screen width is 768px or smaller, the background turns light blue.

üìè Common Breakpoints (Industry Standard)

These breakpoints cover the most common screen sizes:

/* Extra small devices (phones) */ @media (max-width: 576px) { ... } /* Small devices (large phones) */ @media (min-width: 577px) and (max-width: 768px) { ... } /* Medium devices (tablets) */ @media (min-width: 769px) and (max-width: 992px) { ... } /* Large devices (laptops/desktops) */ @media (min-width: 993px) and (max-width: 1200px) { ... } /* Extra large devices (large screens) */ @media (min-width: 1201px) { ... }

üéØ Best Practices for Media Queries

  • ‚úî Use em or rem for font sizes (better for accessibility).
  • ‚úî Start with a "mobile-first" approach (min-width queries).
  • ‚úî Use relative units (%, vh, vw) instead of fixed values (px).
  • ‚úî Don‚Äôt overuse breakpoints‚Äîuse them only when the design breaks.
