Measurement Units

Index

Introduction

In CSS, measurement units can be divided into two main categories: absolute and relative. Let's explore the different types with definitions, examples, and their use cases.

Absolute Units

Absolute units have a fixed value, unaffected by parent elements or the viewport. The most common absolute unit is px.

This box has a fixed width and height of 300px by 150px.

Example

width: 300px; height: 150px;

Use absolute units when you need an element to have a specific fixed size, regardless of the context.

Relative Units

1. % - Percentage

Percentage is relative to the size of the parent element.

This box takes up 50% of its parent’s width and height.

Example

width: 50%; height: 50%;

Use percentages when you want elements to be responsive to their container's size.

2. vw and vh - Viewport Width/Height

These units are relative to the size of the viewport (the visible part of the browser window).

This box is 50% of the viewport width and 20% of the viewport height.

Example

width: 50vw; height: 20vh;

Use vw and vh for layouts that should adapt to the browser window size.

3. em - Relative to Parent Font Size

The unit em is calculated relative to the font size of the parent element.

This text is 2 times the size of its parent’s font size.

Example

font-size: 2em;

Use em when you need font sizes or dimensions to scale relative to their parent container.

4. rem - Relative to Root Font Size

The unit rem is calculated relative to the font size defined on the html element.

This text is 1.5 times the size of the root font size.

Example

font-size: 1.5rem;

Use rem for consistent sizing that scales uniformly across your webpage.

Conclusion

Choosing the right unit depends on your design needs. Use absolute units for fixed sizes, and relative units for responsiveness and flexibility. Experiment with the examples above to see how they work!
